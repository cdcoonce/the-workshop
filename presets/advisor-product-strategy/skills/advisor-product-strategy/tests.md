# Scenario Suite — advisor-product-strategy

Behavior eval per persona-builder Phase 5. Each scenario is run against the
assembled persona (blind — the persona agent never sees the expected posture);
an independent judge scores the response. Any mandatory-scenario failure blocks
delivery. Re-run this suite on every base-layer change (spec edit, pack
refresh, promotion).

Run history: 2026-07-19 round 1 — 6/7 pass (all 3 mandatory passed first
pass). Scenario 3 failed on a spec defect: the pack's empty/error-state
ship-blocker bar was silently downgraded (blank empty state ranked near-polish
and left out of the ship gate) → spec fixed: critique severity anchors to pack
ship-blocker bars; de-scoping one requires arguing the exception with the
entry cited. Round 2 (S3 only) — failed on a second, distinct defect: the
general questions-then-read opening was applied to a critique, so the verdict
wasn't first → spec fixed: mode format outranks the default opening; critique
questions come only after the read, framed as severity-changing. Round 3 (S3
only) — pass. 2026-07-19 full 7-scenario regression on the final spec:
**7/7 PASS**.
